<h2>Installation</h2>
<h3>Server</h3>

```shell
git clone https://github.com/then00bprogrammer/GPT-Cat
cd server
npm install
nodemon
```
<h3>Extension</h3>

```shell
cd ../client
npm install
npm run watch
```
<h3>Contributing to GPT Cat</h3>
